# Vulkan Game Engine

This is a  C++ learning project. It is based on [Little Vulkan Engine](https://github.com/blurrypiano/littleVulkanEngine).

I implemented:
- Entity Component System
- Loading Textures
- AABB
- Drawing Lines
- Integrate GLM
- Integrate ImGui
- Integrate FastGLTF
- Integrate JoltPhysics (WIP)
- Loading GLTF (WIP)
